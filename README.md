# Types of Collisions Inquiry Lab

Interactive cart-collision simulation for teaching momentum, kinetic energy, and restitution.

Includes:

- Fully inelastic, partially inelastic, perfectly elastic, and explosion scenarios
- Inquiry mode and test-your-knowledge mode
- Velocity-time graph with hover/click readout
- Dark and light themes aligned to The Thinking Experiment visual system

Style requirements (persistent):

- Theme toggle stays in the hero card at top-right.
- Light mode follows the Half-Atwood reference style and keeps the left hero accent rail.
- Track and cart colors change between dark and light themes.
- Graph shows cart velocities directly (not probe-only).
- Explosion mode includes a visible pre-release phase before split.
- Layout must avoid hero-card clipping at small widths.
